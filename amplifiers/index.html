Introduction

An amplifier is described as a circuit which is used to increase the magnitude of the signal applied to its input. However, we can’t even justify that all amplifiers are same, they are differentiated according to their circuit configurations and modes of operation.

Generally, Small Signal Amplifiers are commonly used, as they have the ability to amplify the small input signals. For example, a small input signal given to a photo-device will produce a larger output signal which will drive even a relay, lamp or loudspeaker etc.

There are numerous types of electronic circuits defined as amplifiers. Operational Amplifiers, Small Signal Amplifiers, Large Signal and Power Amplifiers are some of the examples of amplifiers. The classification of an amplifier depends upon the size of the signal, i.e, large or small, its physical configuration and how it processes the input signal, i.e, the relationship between input signal and the current flowing in the load.

For simple understanding, we can portray an amplifier as a simple box which contains an amplifying device like a BJT or FET or an OP-AMP, which posses two input terminals and two output terminals (ground being common), and the output signal should be greater than the input signal as it is Amplified.

Ideal Amplifier Model

An ideal signal amplifier will be described as the amplifier whose output signal is linear to the input signal. It has three main properties, they are Input Resistance (RIN), Output Resistance (ROUT) and Gain (A).

Amplifier Gain

The gain of the amplifier can be described as the amplified difference between input and output signals. The Gain is a measure of how much an amplifier “amplifies” the input signal. For example, if we have an input signal of 2 Volts and an output of 20 Volts, then the gain of the amplifier circuit will be “10”, i.e. the output signal is 10 times the input signal; this is called Gain (A).

The gain has no units to represent as it the ratio of the output signal to the input signal, but in Electronics gain is represented by a symbol, A which represents Amplification rather than gain.

There are three different kinds of amplifier gain which can be calculated, these are: Voltage Gain (Av), Current Gain (Ai) and Power Gain (Ap)

For calculating Power Gain you can also divide the power obtained at the output with the power obtained at the input.

The gain of an amplifier is denoted in Decibels (dB). Decibels is the one tenth (1/10th) part of a Bel (B).

To calculate the gain of the amplifier in Decibels or dB, we can use the following expressions.

Voltage Gain in dB: av = 20log(Av) Current Gain in dB: ai = 20log(Ai) Power Gain in dB: ap = 10*log(Ap)

A positive value of dB represents a Gain and a negative value of dB represents a Loss within the amplifier.

For an amplifier, -3 dB is represented as a half-power point, here 0 dB is considered as the maximum output value, so the -3 dB value will be down to maximum value.

Therefore, if an amplifier gain is – 3 dB represents a loss of signal exactly half (x0.5) and if the gain is +3 dB represents a gain of signal exactly double (x2).

Power Amlifiers

The amplifiers are sub classed into two types according to their power or voltage gain, they are Small Signal Amplifier (pre-amplifiers, instrumentation amplifiers etc.) and Large Signal Amplifier (audio power amplifiers, power switching amplifiers etc.).

Small Signal amplifiers are developed to amplify very small voltage levels like microvolts (μV) which are obtained from sensors, audio signals etc. Large signal amplifiers are developed to amplify large input voltage signals or switch heavy load currents, which are mainly observed at driving loudspeakers etc.

These Large Signal Amplifiers are also called as Power Amplifiers, as they are used to deliver power to the load.

Power amplifiers work on the basic principle of converting the DC power drawn from the power supply into an AC voltage signal, which will be delivered to the load. Here the amplification is high, which is a plus point but the efficiency of the conversion from the DC power supply input to the AC voltage signal output is usually poor, which is a main drawback.

Ideal Amplifier

We can now specify the characteristics of an ideal amplifier from our discussion above concerning its Gain:

  • The amplifiers gain, (A) should remain constant for varying values of input signal.
  • Frequency should not effect the gain. Signals of all frequencies must be amplified by exactly the same amount.
  • Noise should not be added by amplifier gain to output signal, even it should remove the noise if it is there in input signal.
  • The temperature should not effect the gain.
  • The gain of amplifier should stay same for long periods of time.

These are the required characteristics of a Ideal Amplifier.

Electronic Amplifier Classes

In Common Emitter Transistor, to operate the transistor in its Active Region some form of Base Biasing is required. A small Base Bias voltage given to input signal will make the transistor to generate the full input waveform at its output with no loss of signal.

Here by altering the Base Bias Voltage, it is possible to operate an amplifier in amplification mode rather than for full waveform generation. By introducing Base Bias voltage to the amplifier, different operating ranges and modes of operation are obtained, these modes of operations are called as Amplifier Class.

These Amplifier Classes are classified in an alphabetical order according to their circuit configurations and mode of operation. These are the designated amplifier classes: class “A”, class “B”, class “C”, and class “AB”. These different amplifier classes range from “a near linear output but with low efficiency” to “a non-linear output but with a high efficiency”.

  • Class A Amplifier – has low efficiency of less than 40% but good signal reproduction and linearity.
  • Class B Amplifier – is twice as efficient as class A amplifiers with a maximum theoretical efficiency of about 70% because the amplifying device only conducts (and uses power) for half of the input signal.
  • Class AB Amplifier – has an efficiency rating between that of Class A and Class B but poorer signal reproduction than Class A amplifiers.
  • Class C Amplifier – is the most efficient amplifier class but distortion is very high as only a small portion of the input signal is amplified, therefore the output signal bears very little resemblance to the input signal. Class C amplifiers have the worst signal reproduction.
